# 测试说明

本目录包含 `lrh-eslint` 项目的测试文件。

## 测试文件结构

- `cli.test.ts` - CLI 命令行接口测试
- `linter.test.ts` - 单元测试，测试 linter 核心功能
- `rules/` - 规则相关的测试文件
  - `index.test.ts` - 规则管理系统测试
  - `no_unused_vars.test.ts` - 未使用变量检测规则测试
  - `semi.test.ts` - 分号检测规则测试

## 运行测试

### 运行所有测试
```bash
npm test
```

### 运行测试并显示覆盖率
```bash
npm run test:coverage
```

### 运行测试（一次性）
```bash
npm run test:run
```
